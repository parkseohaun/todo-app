# todo-app

이 프로젝트는 HTML, JavaScript, Tailwind CSS를 사용한 간단한 할 일 목록 웹앱입니다.

## 기능
- 할 일 추가/삭제
- 마감일 입력 및 D-Day 표시
- 로컬스토리지 저장
- 날짜 유효성 검사
- 할 일 클릭 시 줄긋기

## 사용 방법
1. 웹 페이지를 열고 할 일을 입력하세요.
2. 날짜를 입력한 뒤 추가 버튼을 누르면 등록됩니다.
